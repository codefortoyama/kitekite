投稿ガイドライン

制定日：2026年7月18日

「富山きてきて空港」は、みなさんから寄せられた富山空港の情報で成り立つ非公式ファンサイトです。 気持ちよく情報を共有し合えるよう、投稿の際は以下のガイドラインをご確認ください。

1. 歓迎する投稿

  • 空港の利用レポート・搭乗体験談
  • 空港内のグルメ・お土産のおすすめ
  • 空港へのアクセス方法や駐車場などの便利情報
  • 空港や周辺で開催されるイベントの情報
  • 飛行機や立山連峰がきれいに見えるスポットなど、周辺観光の情報

2. 掲載までの流れ

  • 投稿フォームから送信された内容は、管理者が確認したうえで掲載します。そのまま自動公開されることはありません。
  • 確認・掲載まで日数をいただく場合があります。
  • 内容によっては掲載を見送る場合があります。掲載可否の個別連絡は原則行いません。
  • 掲載にあたって、誤字の修正や長さの調整など軽微な編集を行うことがあります。

3. 投稿時のお願い

  • いつ時点の情報かが分かるように書いてください（フォームの「情報を確認した日」もご活用ください）。
  • 事実と感想・推測は区別して書いていただけると助かります。
  • 料金・営業時間・運航情報などは変わりやすいため、読み手が公式情報を確認する前提での紹介をお願いします。

4. 掲載できない投稿

  • 虚偽の情報、誤解を招く内容
  • 特定の個人・団体への誹謗中傷、プライバシーを侵害する内容（他人が写り込んだ写真の情報等を含む）
  • 著作権・肖像権など、他者の権利を侵害する内容
  • 営利目的の宣伝、スパム
  • 空港の保安上問題となり得る内容（制限区域の詳細等）
  • 政治・宗教に関する主張、その他公序良俗に反する内容

5. 画像について

投稿フォームから直接画像を送ることはできません。ご自身が撮影した（権利を持つ）画像を共有したい場合は、 外部サービスに公開のうえ「参考URL」欄にリンクをご記入ください。

6. 掲載後の修正・削除

ご自身の投稿の修正・削除をご希望の場合は、 トップページの投稿フォーム、または GitHub リポジトリの Issue からご連絡ください。

7. 投稿の取り扱い

投稿内容の著作権は投稿者に帰属します。投稿をもって、当サイト上での掲載（前述の軽微な編集を含む）を 無償で許諾いただいたものとします。

当サイトは富山空港・航空会社・行政機関・交通事業者とは関係のない非公式サイトです。 本ガイドラインは必要に応じて改定することがあります。
